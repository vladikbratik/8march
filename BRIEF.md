# ТЗ: Лендинг-открытка «8 марта» — [НАЗВАНИЕ ФИЛЬМА]

## Стек

- **Vite** (сборщик, vanilla JS)
- **GSAP** + **ScrollTrigger** (все анимации)
- **CSS** (чистый, без Tailwind — нужна полная свобода)
- Деплой: **Vercel** (`vercel deploy`)
- Никаких фреймворков (React/Vue не нужны — статичная страница)

---

## Структура проекта

```
/project
  index.html
  /src
    main.js          — GSAP анимации, логика
    style.css        — все стили
  /assets
    /video
      main.mp4       — одно общее видео (загружается позже)
    /photos
      01_shustova.jpg
      02_shurako va.jpg
      ... (по каждой сотруднице)
    /fonts

  /public
    og-image.jpg     — превью для Bitrix/Telegram/WhatsApp (1200×630px)

  package.json
```

## Визуальная концепция

**Кино-арт-хаус.** Минимализм, атмосфера, зерно плёнки.

### Палитра
```
--bg:        #1a1612   /* почти чёрный, тёплый */
--cream:     #f5f0e8   /* кремовый для текста */
--accent:    #c9a96e   /* золото, приглушённое */
--mid:       #3d3530   /* промежуточный фон */
```

### Типографика
- Заголовки, имена: `Cormorant Garamond` (serif, элегантный)
- Должности, подписи: `Inter` (sans-serif, нейтральный)

### Текстура
Поверх всей страницы — псевдоэлемент `::before` на `body` с SVG-шумом (grain), opacity 0.04. Создаёт ощущение плёнки.

## Секции лендинга (сверху вниз)

---

### СЕКЦИЯ 1 — Hero / Заставка

**Поведение:** При загрузке страница чёрная. Медленно (2 сек) появляется текст — как начало арт-хаусного фильма.

**Контент:**
```
[маленький текст сверху, letter-spacing широкий]
PANDA AUTOMATIC PRESENTS

[большой заголовок, Cormorant Garamond, ~80px]
[НАЗВАНИЕ ФИЛЬМА]

[подзаголовок]
8 марта 2026

[маленький текст снизу, fade in с задержкой]
↓ scroll
```

**Анимация:**
- `gsap.from` на каждый элемент, stagger 0.4s, y: 20 → 0, opacity: 0 → 1
- Индикатор скролла пульсирует

---

### СЕКЦИЯ 2 — Видео

**Поведение:** При скролле к секции — видео плавно появляется (fade in). Видео с рамкой как кинокадр — тонкая белая рамка, лёгкое виньетирование через CSS.

**Контент:**
- `<video>` тег, src: `/assets/video/main.mp4`
- `autoplay muted playsinline loop`
- Над видео маленький текст: `реж. коллектив`
- Под видео маленький текст: `2026`

**Вёрстка:**
- Видео max-width: 860px, centered
- Соотношение сторон 16:9
- Рамка: `border: 1px solid rgba(255,255,255,0.15)`
- Виньетка: box-shadow inset `0 0 80px rgba(0,0,0,0.6)`

### СЕКЦИЯ 4 — Финальный кадр

**Поведение:** Экран плавно темнеет до полного чёрного (как fade out в кино). Затем поочерёдно появляются строки белого текста — как финальные титры.

**Анимация:**
- Overlay `rgba(0,0,0,1)` появляется через ScrollTrigger когда секция входит в viewport
- Затем gsap timeline: каждая строка появляется с задержкой 0.6s

**Контент:**
```
[маленький текст]
продолжение следует

[большой текст, Cormorant, ~60px]
6 марта

[средний текст]
[ВРЕМЯ] • 1 этаж

[маленький текст, opacity 0.5, после паузы]
С праздником.
Panda Automatic
```

## OG-теги (для Bitrix / Telegram / WhatsApp)

В `<head>` index.html:

```html
<meta property="og:title" content="[НАЗВАНИЕ ФИЛЬМА] — 8 марта 2026" />
<meta property="og:description" content="Поздравление от коллег. 6 марта, 1 этаж." />
<meta property="og:image" content="https://[домен]/og-image.jpg" />
<meta property="og:type" content="website" />
<meta name="twitter:card" content="summary_large_image" />
```

`og-image.jpg` — создать отдельно: тёмный фон, название фильма по центру, логотип Panda Automatic, размер 1200×630px.

## Порядок разработки для Claude Code

1. Scaffold проекта: `npm create vite@latest . -- --template vanilla`
2. Установить зависимости: `gsap`
3. Создать структуру папок
4. Написать `index.html` с разметкой всех секций и OG-тегами
5. Написать `style.css` — базовые стили, палитра, типографика, grain-текстура
6. Написать `main.js` — все GSAP анимации секция за секцией
7. Заполнить данные сотрудниц (имена + посвящения уже в этом файле)
8. Плейсхолдеры для фото и видео (добавятся позже)
9. Проверить на мобильном (media query breakpoint: 768px)
10. `vercel deploy`
